# 中文打字练习

一个在浏览器里运行的**中文打字练习**网页应用 —— 用**拼音输入法**练习打字，统计字/分钟与准确率。无需账号、无服务器、不上传任何数据。

本项目基于开源项目 [type-review](https://github.com/xiaolai/type-review)（作者 xiaolai，MIT 许可证）改造而来：把英文语料换成中文古诗文与名句，把逐键匹配改成支持**拼音输入法**的逐字匹配，并相应调整了界面、指标（字/分钟）与统计；移除了原来针对英文 26 个字母设计的「字母解锁 / 分指统计 / 屏幕键盘」等功能。

## 功能

- **拼音输入打字** —— 屏幕显示一段中文，你用系统拼音输入法逐字打出来，应用逐字比对（支持一次上屏多个字）。
- **两种练习方式** —— 「按字数」打完固定长度的一段；「按时间」在限定时长内自由地一直打。
- **真实的文字** —— 内置一批公有领域的古诗文与名句；也可以在「文库」里粘贴 / 上传自己的 `.txt` / `.md`，或在练习页用「自定义文本」临时练一段。
- **实时指标** —— 字/分钟、准确率、稳定性，以及每秒速度曲线、回放、分享链接。
- **统计** —— 速度趋势、练习日历、连续天数、里程碑等，全部按字/分钟计。
- **四种主题 + 机械键盘音效**，可选显示空白符。
- **数据全在本机** —— 用 IndexedDB 存储，一键导出 JSON 备份 / 导入 / 清空。

## 本地运行

需要 Node 18+（已在 Node 22 上验证）。

```bash
npm install
npm run dev        # 开发服务器，默认 http://localhost:5173
```

打开后**确认系统输入法切到拼音**，照着屏幕上的文字打即可。

其它脚本：

```bash
npm run build      # 类型检查 + 生产构建，产物在 dist/
npm run preview    # 本地预览构建产物
npm test           # 运行测试（Vitest）
npm run typecheck  # 仅类型检查
npm run lint       # Biome 检查
```

> 注：项目自带 `pnpm-lock.yaml`，若你已装 pnpm 也可用 `pnpm install` / `pnpm dev` 等同名脚本。

## 部署

`npm run build` 后把 `dist/` 目录作为静态站点托管即可（Cloudflare Pages、Vercel、Netlify、Nginx 等都可以），无需后端。

## 键盘快捷键

- **拼音输入法** —— 直接打字，应用捕捉上屏的汉字。
- **Tab** —— 换一段新文字。
- **Enter** —— 在结果页直接开始下一段。
- **退格** —— 修正已上屏的字（盲打模式下禁用）。

## 技术栈

SolidJS + TypeScript · Vite · Vitest · Biome。唯一的运行时依赖是 `solid-js`。
